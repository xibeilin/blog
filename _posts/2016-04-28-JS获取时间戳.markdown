## JS 获取时间戳

JavaScript 获取当前时间戳：


1. 第一种方法：

	``` 
	var timestamp =Date.parse(new Date());
	```
	
	结果：1280977330000


2. 第二种方法：

	```
	var timestamp =(newDate()).valueOf();
	```
	
	结果：1280977330748

3. 第三种方法：

	``` 
	var timestamp=newDate().getTime()；
	```
	结果：1280977330748

第一种：获取的时间戳是把毫秒改成000显示，

第二种和第三种是获取了当前毫秒的时间戳。



### 注意:

JS 的时间戳 和PHP 的是不太一样的,

JS 的包括毫秒数,是13位的,

PHP 的是10位的.

所以转换时要注意!
